benice adalah program CLI sederhana menggunakan bahasa Python, program ini dibuat untuk memeriksa kekuatan password.
<br>

Beberapa pengecekan yang dilakukan oleh benice:
- Huruf besar
- Huruf kecil 
- Simbol unik 
- Angka 
- Minimal panjang 6
<br>

Tingkatan keamanan password terdiri:
- Weak
- Medium
- Strong
